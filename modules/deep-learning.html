What's happening under the hood

Each training step runs a forward pass (inputs flow right, each neuron computing a weighted sum + activation), measures the loss, then backpropagates the error to nudge every weight downhill.

# forward
z = W·a + b
a' = σ(z)

# update
W ← W − η ∂L/∂W
